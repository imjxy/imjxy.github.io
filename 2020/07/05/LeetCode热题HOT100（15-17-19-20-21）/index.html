15三数之和

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组

示例：

这道题是双指针的一个使用，要找三个数，就for一下每个数用双指针，由于要求不含有重复三元组，首先排序一下，在指针前进的时候比较是否走过，如果跟上一个重复就跳过。

19删除链表的倒数第N个节点

给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

示例：

给定一个链表: 1->2->3->4->5, 和 n = 2.

当删除了倒数第二个节点后，链表变为 1->2->3->5.
说明：

给定的 n 保证是有效的。

这个题很巧的一个地方在于你可以使用快慢指针，先让快指针走n步，然后在让快慢指针一起走，当快指针走到末端时，慢指针就指向了需要删除的结点。这个题我一开始快慢指针都指向了给定链表的头结点，然后完成删除操作后返回头结点，但是在删除头结点或者是最后一个节点时会出现问题，于是就先搞一个节点指向头结点，快慢指针都从头结点之前出发，这样也保证了慢指针最后指向的是目标节点的前一个，便于删除。

20有效的括号

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: “()”
输出: true

这个题就是栈的运用了，首先用哈希表存储左右括号为键值，遇到键（左括号）就进栈，遇到值（右括号）就出栈，如果不匹配就提前返回false，最后判断栈是否为空，为空则说明成功匹配。而且在过程中，如果栈的size超过了字符串大小的一半，可以直接返回false，因为不论如何都不可能最后为空了。

21合并两个有序链表

将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

示例：

输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
